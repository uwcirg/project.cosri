Launch COSRI demo system
Launch Instructions:
  1. Register yourself as a user (if not previously registered)
  2. You’ll be taken to a “SMART App Launcher” page
  3. In the bottom section on the page called “Launch”, click the “Launch App” button
  4. You’ll be taken to a “Provider Login” page - use the credentials already filled in on the text fields and click Login
  5. You’ll be taken to the COSRI dashboard

Notes:
- Demo system accesses a test Prescription Drug Monitoring Program (PMP) server through the Washington Health Information Exchange. The queries and pathways are real, but the data are not. - We use a UW-customized, UW-hosted version of the open-source “SMART App Launcher” to run the demo system outside an Electronic Health Record (EHR). We have not yet changed the branding.
